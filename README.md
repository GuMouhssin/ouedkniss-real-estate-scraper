# OuedKniss Real Estate Scraper

A Python-based GraphQL scraper for extracting Algerian real estate listings from OuedKniss.

## Features

- GraphQL API scraping
- Dynamic pagination
- Retry system
- Structured parsing
- Semi-structured data normalization
- CSV export
- JSON export
- Data cleaning

## Technologies

- Python
- Requests
- GraphQL
- JSON
- CSV
